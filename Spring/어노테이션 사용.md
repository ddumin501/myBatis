
Spring Bean Configuration File (.xml) 대신 class(.java) 사용하기
```java
//Beans.java
@Configuration
public class Beans {
	@Bean
	public First first() {
		// return new First();
		First f = new First();
		f.setMsg("금요일입니다");
		return f;
	}

}
//bea
ApplicationContext ctx = new AnnotationConfigApplicationContext(Beans.class);
		First first = ctx.getBean("first", a.First.class);
```
