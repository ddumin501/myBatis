

Spring은 객체를 싱글톤(Singleton) 객체형식으로 관리해준다.

## 의존성 주입관계란?

```java
public interface B{
	String b();
}

A a = new A(); 
a.setB(new B1());
sop(a); //"b1"

Spring에서는 
//A a = new A();  --> XML파일로 설정
//a.setB(new B1());
sop(a); //"b1"
```
